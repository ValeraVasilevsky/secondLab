Задача 8
Реализуйте систему вывода текстовых сообщений. Базовый компонент умеет просто выводить сообщение в консоль. С помощью декораторов нужно добавить возможность:
1.	Шифровать сообщение (например, сдвигом символов по ASCII — шифр Цезаря).
2.	Добавлять отметку времени перед сообщением.
3.	Записывать сообщение не только в консоль, но и в файл.
Каждый из этих декораторов можно комбинировать — например, сообщение может быть зашифровано, снабжено временем и выведено в файл одновременно.
Напишите демонстрационную программу, использующую разработанные классы.
